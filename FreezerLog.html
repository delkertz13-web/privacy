Politique de Confidentialité - FreezerLog

Dernière mise à jour : 9 mars 2026

1. Introduction

Bienvenue dans FreezerLog. Nous prenons votre confidentialité très au sérieux. Cette Politique de Confidentialité explique comment FreezerLog traite vos données.

2. Aucune Collecte de Données

FreezerLog ne collecte, n'enregistre, ni ne transmet aucune donnée personnelle.

FreezerLog fonctionne entièrement en mode hors ligne. Aucune information sur vos inventaires de congélateur, vos listes d'articles ou vos préférences ne quitte votre appareil.

3. Stockage Local

Toutes les données de FreezerLog sont stockées localement sur votre appareil iOS. Ces données incluent :

  • Vos inventaires de congélateur
  • Les articles et leur contenu
  • Les dates d'expiration
  • Vos préférences d'application
  • Toute autre information saisie dans l'application

Ces données sont entièrement sous votre contrôle et restent privées sur votre appareil.

4. Aucune Transmission de Données

FreezerLog :

  • N'envoie aucune donnée à des serveurs externes
  • N'établit aucune connexion réseau pour synchroniser ou partager vos données
  • N'effectue aucun suivi analytique
  • N'utilise aucun outil de publicité tiers
  • N'utilise aucun service de crash reporting associé à vos données personnelles

5. Sécurité

Votre sécurité dépend de la sécurité de votre appareil iOS. Nous vous recommandons :

  • De maintenir votre appareil à jour avec les dernières mises à jour iOS
  • D'utiliser un code d'accès ou Face ID/Touch ID pour protéger votre appareil
  • De sauvegarder régulièrement vos données via iCloud ou iTunes

6. Suppression de Données

Vous avez un contrôle total sur vos données. Vous pouvez :

  • Supprimer n'importe quel article ou inventaire dans l'application
  • Supprimer complètement l'application et toutes ses données en la désinstallant
  • Restaurer vos données depuis une sauvegarde iCloud ou iTunes si nécessaire

7. Permissions de l'Application

FreezerLog ne demande aucune permission spéciale au-delà de celles nécessaires pour le fonctionnement de base de l'application.

8. Modifications de cette Politique

Nous nous réservons le droit de modifier cette Politique de Confidentialité à tout moment. Les modifications seront communiquées via une mise à jour de l'application. Votre utilisation continue de FreezerLog après la publication des modifications constitue votre acceptation des modifications.

9. Nous Contacter

Si vous avez des questions concernant cette Politique de Confidentialité ou les pratiques de FreezerLog en matière de confidentialité, veuillez nous contacter.

FreezerLog : Votre congélateur, vos données, votre vie privée.
