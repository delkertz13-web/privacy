Questions Fréquemment Posées

Comment puis-je restaurer mes achats premium ?
Si vous avez précédemment acheté la version premium de DryCount sur cet appareil ou un autre appareil Apple avec le même identifiant Apple, vous pouvez restaurer votre accès en appuyant sur le bouton "Restaurer les achats" dans l'onglet Paramètres de l'application. Cette action est gratuite et instantanée. Assurez-vous d'être connecté avec le même identifiant Apple utilisé pour l'achat initial.
Comment puis-je supprimer mes données personnelles ?
Vous pouvez supprimer toutes vos données de DryCount (journal, streaks, statistiques) à tout moment en accédant à l'onglet Paramètres et en sélectionnant "Supprimer toutes les données". Cette action est irréversible et supprimera définitivement vos informations de l'appareil. Pour supprimer vos données de nos serveurs de sauvegarde iCloud, vous devrez également accéder à Réglages > [Votre nom] > iCloud > Gérer le stockage iCloud et supprimer DryCount.
Qu'est-ce qui est inclus dans la version premium ?
La version premium de DryCount déverrouille des statistiques avancées, y compris des graphiques détaillés de consommation, l'analyse des tendances sur plusieurs mois, l'export de rapports PDF mensuels, et un widget iOS avancé pour votre écran d'accueil. Le suivi quotidien, les streaks et les jalons de santé restent gratuits pour tous les utilisateurs.
Puis-je réinitialiser mon compteur de jours consécutifs ?
Oui, vous pouvez réinitialiser votre streak directement depuis l'onglet Dashboard. Appuyez et maintenez sur votre compteur actuel, puis confirmez votre intention de réinitialiser. Cela réinitialisera votre compteur à 0 jours mais conservera tout votre historique de journal pour votre référence future. Vous pouvez commencer un nouveau streak immédiatement après.
Mes données sont-elles sauvegardées sur iCloud ?
Oui, DryCount utilise iCloud CloudKit pour sauvegarder automatiquement votre journal, vos streaks et vos paramètres lorsque vous êtes connecté avec votre identifiant Apple. Cela signifie que vos données se synchronisent entre vos appareils Apple et restent sûres. Vous pouvez désactiver cette synchronisation à tout moment dans les paramètres de l'application, mais vos données seront alors stockées localement uniquement.

Restaurer les Achats

Si vous avez acheté la version premium de DryCount, vous pouvez restaurer votre accès à tout moment. Cette fonctionnalité vous permet de conserver votre premium sur plusieurs appareils ou après une réinstallation de l'application.

Remarque : Vous devez être connecté avec le même identifiant Apple utilisé pour l'achat initial. La restauration est gratuite et instantanée.

Supprimer Mes Données

Vous pouvez supprimer toutes vos données personnelles (journal, streaks, statistiques, préférences) de DryCount. Cette action est irréversible et supprimera définitivement vos informations de votre appareil.

Attention : Cette action ne peut pas être annulée. Assurez-vous de vouloir vraiment supprimer toutes vos données avant de continuer. Pour une meilleure protection, envisagez d'exporter un rapport PDF avant suppression.
