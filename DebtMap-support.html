Questions Fréquemment Posées

Quelle est la différence entre la méthode Snowball et Avalanche ?
Snowball : Rembourse d'abord les petites dettes (psychologiquement motivant avec des victoires rapides).

Avalanche : Rembourse d'abord les dettes au taux d'intérêt le plus élevé (économise le plus d'argent sur le long terme).

DebtMap calcule et visualise les deux stratégies pour vous aider à choisir celle qui vous convient.
Mes données sont-elles sécurisées ?
Oui, absolument. DebtMap fonctionne 100% hors ligne. Aucune donnée personnelle n'est envoyée à un serveur, stockée en cloud ou partagée. Vos informations financières restent uniquement sur votre téléphone, sous le contrôle total de l'App Store et des protections de sécurité d'iOS.
Puis-je simuler un paiement supplémentaire chaque mois ?
Oui ! DebtMap dispose d'un simulateur "Et si je payais X€ de plus par mois ?" qui montre instantanément l'impact sur votre date de libération totale. C'est un excellent outil pour voir combien de temps vous gagnerez en augmentant vos paiements.
Comment fonctionne le calcul des intérêts ?
DebtMap utilise un calcul d'intérêt composé basé sur le taux annuel que vous entrez, appliqué mensuellement. Le montant des intérêts est recalculé chaque mois en fonction du solde restant et de votre paiement mensuel, ce qui refléte la réalité de vos dettes.
Puis-je récupérer mon achat si je change de téléphone ?
Oui. Utilisez la fonction "Restaurer les achats" (voir ci-dessous) sur votre nouveau téléphone avec le même compte Apple. Si vous avez acheté DebtMap, vous n'aurez pas à payer à nouveau. C'est garanti par le système d'App Store.

Restaurer l'Achat

Si vous avez acheté DebtMap sur un autre appareil ou après une réinstallation, utilisez cette option pour restaurer votre accès sans frais supplémentaires.

Comment faire :
  1. Ouvrez DebtMap
  2. Allez dans Paramètres (icône roue)
  3. Appuyez sur "Restaurer les achats"
  4. Connectez-vous avec le même identifiant Apple

Vous serez redirigé vers l'App Store où vous pouvez voir l'historique de vos achats et les gérer.

Supprimer Mes Données

DebtMap stocke toutes vos données localement sur votre appareil. Vous pouvez les supprimer à tout moment.

⚠️ Important : La suppression de vos données est irréversible. Tous vos profils de dettes, simulations et historique seront perdus. Assurez-vous de vraiment vouloir procéder avant de continuer.

Option 1 : Supprimer depuis l'app

Allez dans Paramètres → Gestion des données → Supprimer toutes mes données

Option 2 : Désinstaller l'app

Appuyez longuement sur l'icône DebtMap → Supprimer l'app → Supprimer l'app

Option 3 : Réinitialiser l'iPhone

Réglages → Général → Transférer ou réinitialiser → Réinitialiser → Effacer le contenu et les réglages

Cette action est irréversible.
