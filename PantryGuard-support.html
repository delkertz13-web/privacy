Questions Fréquemment Posées

Comment ajouter des articles rapidement à mon inventaire ? +
PantryGuard propose trois méthodes d'ajout rapide : vous pouvez taper le nom de l'article, scanner le code-barres avec votre appareil photo, ou utiliser la reconnaissance vocale. Les codes-barres sont résolus hors ligne localement pour une recherche instantanée sans connexion Internet.
Que signifie l'alerte "J-3" sur les produits ? +
L'alerte "J-3" signifie que le produit expirer dans 3 jours. Les articles avec une alerte active apparaissent en priorité dans la liste "À utiliser en priorité" et dans les notifications du widget. Cela vous permet de planifier vos repas pour utiliser les aliments avant qu'ils ne s'abîment.
Comment utiliser le widget sur mon écran d'accueil ? +
Maintenez votre écran d'accueil enfoncé, appuyez sur le bouton plus, cherchez PantryGuard et sélectionnez le widget. Le widget affiche les 3 produits qui périment bientôt directement sur votre écran d'accueil pour un accès rapide. Vous pouvez ajouter plusieurs instances du widget si nécessaire.
Quelle est la différence entre la version gratuite et PantryGuard Pro ? +
La version gratuite vous permet de gérer jusqu'à 30 articles. PantryGuard Pro offre un nombre illimité d'articles, des catégories personnalisées pour organiser votre inventaire, et un widget configurable avec vos préférences. C'est un achat unique à 2,99$ sans abonnement récurrent.
Comment fonctionne le calcul des statistiques de gaspillage évité ? +
PantryGuard estime les économies en comparant les articles marqués comme "utilisés" avec le prix moyen des produits alimentaires. Chaque mois, vous verrez une statistique de gaspillage évité qui vous montre combien d'argent vous avez économisé en évitant les achats doublons et en utilisant les aliments avant expiration.

Restaurer un Achat

Si vous avez acheté PantryGuard Pro sur un autre appareil ou après une réinstallation, vous pouvez restaurer votre achat. Cette action rechargera votre licence depuis l'App Store.

Vous serez invité à vous connecter avec votre identifiant Apple.

Supprimer les Données

Vous pouvez supprimer toutes vos données PantryGuard (inventaire, historique, paramètres) de votre appareil. Cette action est irréversible.
