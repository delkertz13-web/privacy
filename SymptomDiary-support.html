Questions Fréquentes

Comment exporter mes données en PDF pour mon médecin?
▼
Dans l'onglet "Rapports", sélectionnez la période souhaitée (30, 90 jours ou personnalisée). Appuyez sur "Générer Rapport PDF" et choisissez de partager le fichier par email ou iCloud. Le rapport contient un résumé structuré de tous vos symptômes, leurs patterns et les corrélations identifiées.
Où sont stockées mes données et sont-elles sécurisées?
▼
Toutes vos données sont stockées localement sur votre iPhone. Aucune information n'est envoyée à des serveurs externes. Vos données restent sous votre contrôle total. Si vous utilisez iCloud, vous pouvez activer la synchronisation optionnelle uniquement entre vos appareils personnels via le chiffrement de bout en bout d'Apple.
Comment puis-je supprimer ou modifier une entrée de symptôme?
▼
Dans la vue Timeline, balayez vers la gauche sur l'entrée que vous souhaitez supprimer, ou appuyez sur "Modifier" pour ajuster les détails (intensité, déclencheurs, durée). Vous pouvez aussi appuyer longuement sur une entrée pour voir plus d'options. Toutes les modifications sont enregistrées automatiquement.
Puis-je tracker plusieurs types de symptômes à la fois?
▼
Oui, vous pouvez tracker simultanément douleurs, fatigue, migraines, troubles digestifs, et plus. Le widget d'accueil permet une saisie rapide en 10 secondes. La Timeline affiche tous vos symptômes avec un code couleur pour identifier facilement les patterns et les corrélations entre différents symptômes.
Comment fonctionne la restauration d'achat?
▼
Si vous avez déjà acheté l'accès illimité et que vous avez réinstallé l'app ou l'utilisez sur un nouvel appareil, utilisez le bouton "Restaurer Achats" pour récupérer votre accès. Vous serez invité à vous connecter avec votre identifiant Apple qui a effectué l'achat initial.
