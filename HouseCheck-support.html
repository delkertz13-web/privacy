Questions fréquemment posées

🆓 Quelles tâches de maintenance sont incluses dans la version gratuite ?
La version gratuite de HouseCheck inclut 20 tâches de maintenance saisonnières pré-remplies (Printemps, Été, Automne, Hiver). Vous obtenez un calendrier complet optimisé pour les propriétaires américains. La version Pro ($2.99/an) déverrouille les tâches personnalisées, l'historique illimité et l'export PDF.
📍 Comment les rappels sont-ils ajustés selon ma zone climatique ?
Durante l'onboarding, vous entrez votre code postal. HouseCheck utilise les zones de rusticité USDA pour adapter les dates de maintenance (par exemple, l'hivernage des tuyaux apparaît plus tôt dans les zones froides). Le calendrier saisonnier s'ajuste automatiquement selon votre région.
🔒 Mes données sont-elles sauvegardées en ligne ou en local ?
HouseCheck fonctionne intégralement hors ligne. Toutes vos données de tâches, historique et contacts sont stockés localement sur votre iPhone. Aucun serveur, aucun compte requis. Vos informations de propriété restent privées.
📄 Comment exporter mon historique pour la vente de ma maison ?
La version Pro permet d'exporter votre historique complet de maintenance en PDF. Ce document devient utile pour prouver l'entretien régulier lors de la vente de votre maison. Incluez-le dans votre dossier de propriété ou partagez-le avec les acheteurs potentiels.
📱 Puis-je utiliser HouseCheck sans créer de compte ?
Oui ! HouseCheck ne nécessite absolument aucun compte. Tout fonctionne immédiatement. L'onboarding simple (type de logement, année de construction, code postal) prend 60 secondes. Vos données ne quittent jamais votre iPhone.

Restaurer achats

Si vous avez acheté HouseCheck Pro sur un autre appareil ou après une réinstallation, restaurez votre abonnement.

Vous serez invité à vous connecter avec votre Apple ID pour confirmer l'achat.

Supprimer les données

Supprimez toutes vos données HouseCheck de cet appareil. Cette action ne peut pas être annulée.

⚠️ Attention : La suppression effacera définitivement tous vos tâches, historique de maintenance, contacts et paramètres. Assurez-vous d'avoir exporté vos données importantes avant de continuer.

Cette action supprimera uniquement les données locales. Votre compte App Store n'est pas affecté.

À propos de HouseCheck

HouseCheck aide les propriétaires américains à ne jamais oublier une tâche de maintenance essentielle. Conçu pour les primo-accédants et propriétaires récents qui veulent éviter les réparations coûteuses.

  • Calendrier saisonnier automatique (Spring, Summer, Fall, Winter)
  • Rappels basés sur votre zone climatique (code postal USDA)
  • Historique avec photos et coûts
  • Carnet de contacts (plombier, électricien, HVAC)
  • Fonctionne complètement hors ligne
  • Aucun compte ou inscription requis
