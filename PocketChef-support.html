❓ Questions Fréquemment Posées

1. Comment ajouter mes propres recettes ?
Accédez à l'onglet "Mes Recettes" et appuyez sur le bouton "+". Remplissez les détails de votre recette, ajoutez une photo, listez les ingrédients et les instructions. Vous pouvez également ajouter des notes personnalisées pour chaque recette.
2. Comment planifier mes repas pour la semaine ?
Accédez à l'onglet "Plan de Repas" et sélectionnez une recette pour chaque jour de la semaine. PocketChef générera automatiquement une liste d'ingrédients consolidée pour tous vos repas planifiés. Vous pouvez modifier votre plan à tout moment.
3. Puis-je rechercher des recettes par ingrédient ?
Oui ! Utilisez la barre de recherche dans l'onglet "Recettes". Tapez un ingrédient et l'application affichera toutes les recettes contenant cet ingrédient. Vous pouvez aussi filtrer par catégories comme Petit-déjeuner, Déjeuner, Dîner, Dessert, etc.
4. Qu'est-ce que la version Premium offre ?
La version Premium PocketChef propose : recettes illimitées, plans de repas illimités, thèmes supplémentaires, notes avancées, export de listes d'ingrédients en PDF, et synchronisation cloud. L'application gratuite permet 50 recettes et 12 plans de repas par mois.
5. Mes données sont-elles sauvegardées ?
Oui, vos données sont stockées localement sur votre appareil iOS. Si vous avez activé iCloud dans les paramètres de PocketChef, vos recettes et plans de repas seront synchronisés sur tous vos appareils Apple connectés au même compte iCloud.

🔄 Restaurer les Achats

Si vous avez acheté la version Premium PocketChef sur un autre appareil ou après avoir réinstallé l'application, vous pouvez restaurer votre abonnement.

À savoir : Les achats intégrés dans l'App Store d'Apple sont toujours associés à votre identifiant Apple.

Comment restaurer votre achat :

  1. Ouvrez PocketChef
  2. Allez dans Paramètres (icône d'engrenage)
  3. Appuyez sur "Abonnement"
  4. Appuyez sur "Restaurer les Achats"
  5. Connectez-vous avec votre identifiant Apple si nécessaire
  6. Votre abonnement Premium sera automatiquement restauré
Problème : Si la restauration ne fonctionne pas, vérifiez que vous utilisez le même identifiant Apple que celui utilisé pour l'achat initial.

🗑️ Supprimer les Données

Vous avez la possibilité de supprimer vos données personnelles de PocketChef.

Supprimer des données spécifiques :

Allez dans Paramètres > Confidentialité et Données. Vous pouvez supprimer :

  • Recettes individuelles
  • Plans de repas
  • Toutes les notes personnalisées

Supprimer toutes les données :

  1. Allez dans Paramètres > Confidentialité et Données
  2. Appuyez sur "Supprimer toutes les données"
  3. Confirmez en appuyant sur "Supprimer définitivement"
  4. Toutes vos recettes, plans de repas et notes seront supprimés
Attention : Cette action est irréversible. Assurez-vous que vous avez sauvegardé les recettes que vous souhaitez conserver avant de procéder.

Demander la suppression de compte :

Pour demander une suppression complète de votre compte et de vos données personnelles (conformément au RGPD), envoyez un email à delkertz13@gmail.com avec le sujet "Suppression de compte PocketChef".

ℹ️ Informations Supplémentaires

Paramètres Recommandés :

  • Notifications : Activez les notifications de rappel pour vos plans de repas
  • Stockage iCloud : Activez la synchronisation pour accéder à vos recettes sur tous vos appareils
  • Photos : Autoriser l'accès aux photos pour ajouter des images à vos recettes

Version de l'Application :

Pour vérifier votre version de PocketChef, allez dans Paramètres > À Propos. Assurez-vous que vous avez la dernière version disponible sur l'App Store pour bénéficier de toutes les fonctionnalités et corrections de bugs.
