Questions fréquemment posées

Trouvez les réponses aux questions les plus posées sur PocketChef.

Comment ajouter une nouvelle recette ?
Appuyez sur le bouton "+" dans l'onglet Recettes, remplissez le nom et les ingrédients, puis ajoutez une photo si vous le souhaitez. Vous pouvez également ajouter des notes personnalisées et des instructions détaillées. La recette sera sauvegardée automatiquement dans votre collection personnelle.
Comment fonctionne la planification de repas ?
Accédez à l'onglet "Planification", sélectionnez les jours de la semaine, puis choisissez vos recettes pour chaque repas. PocketChef génère automatiquement une liste d'ingrédients consolidée basée sur vos repas planifiés. Vous pouvez modifier et organiser votre liste avant vos courses.
Puis-je synchroniser mes recettes entre appareils ?
Avec la version Premium, vos recettes et plans de repas sont synchronisés automatiquement sur tous vos appareils iOS via iCloud. La version gratuite stocke les données localement sur votre appareil.
Comment rechercher une recette par ingrédient ?
Utilisez la barre de recherche en haut de l'onglet Recettes. Vous pouvez rechercher par nom de recette, catégorie ou ingrédient spécifique. PocketChef affiche toutes les recettes correspondantes instantanément.
Quelle est la limite de recettes gratuites ?
La version gratuite vous permet de créer jusqu'à 25 recettes personnelles et de planifier 4 semaines de repas. Passez à Premium pour un nombre illimité de recettes, des thèmes personnalisés, et plus de fonctionnalités avancées.

Restaurer les achats

Si vous avez déjà acheté la version Premium mais que celui-ci n'est pas actif, vous pouvez le restaurer.

Vous devez être connecté avec le même identifiant Apple que celui utilisé pour l'achat initial.

Besoin d'aide ?

Si la restauration ne fonctionne pas, veuillez :

  • Vérifier que vous êtes connecté avec le bon identifiant Apple
  • Vérifier votre connexion Internet
  • Redémarrer l'app
  • Nous contacter si le problème persiste
