Questions Fréquemment Posées

Comment ajouter un achat à ReturnPal?
Appuyez sur le bouton "+" pour ajouter un nouvel achat. Entrez le nom du retailer, la description de l'article, la date d'achat et la durée de la fenêtre de retour. Si votre retailer figure dans notre bibliothèque (Amazon, ASOS, Zara, etc.), la durée sera pré-remplie automatiquement. Vous pouvez aussi la modifier manuellement si nécessaire.
À quel moment recevrai-je des notifications?
ReturnPal vous envoie des notifications intelligentes pour vous aider à ne pas rater vos retours:
  • 7 jours avant expiration
  • 3 jours avant expiration
  • 1 jour avant expiration
Vous pouvez personnaliser ces préférences dans les paramètres de l'app.
Puis-je utiliser ReturnPal gratuitement?
Oui! ReturnPal fonctionne en mode freemium. La version gratuite vous permet de suivre jusqu'à 10 achats actifs simultanément. Si vous avez besoin de suivre plus d'achats, vous pouvez passer à ReturnPal Pro pour $1.99 (achat unique), qui déverrouille un nombre illimité d'achats, une bibliothèque retailers complète et l'accès à votre historique archivé.
Comment la timeline visuelle fonctionne-t-elle?
L'écran principal de ReturnPal affiche une timeline visuelle de tous vos achats, triés par urgence de retour. Vous voyez le nombre exact de jours restants pour chaque article, avec une mise en évidence visuelle pour les articles dont le délai approche. Un widget d'écran d'accueil vous permet aussi de voir rapidement vos retours les plus urgents sans ouvrir l'app.
ReturnPal peut-il m'amener directement à l'app de mon retailer?
Absolument! Si l'app du retailer est installée sur votre iPhone (Amazon, ASOS, etc.), ReturnPal peut ouvrir un deep link directement vers la page de retour appropriée. Cela accélère le processus et vous évite de chercher manuellement votre commande.

Gestion de Compte

Restaurer l'Achat

Si vous avez déjà acheté ReturnPal Pro mais que vous l'utilisez sur un nouvel appareil, vous pouvez restaurer votre achat sans avoir à payer à nouveau.

Note: Cette action nécessite de vous connecter avec votre Apple ID associé à l'achat original.

Supprimer mes Données

Si vous souhaitez supprimer définitivement toutes vos données personnelles stockées dans ReturnPal, vous pouvez le faire à tout moment. Cette action est irréversible.

Attention: La suppression de vos données effacera tous vos achats suivis, votre historique et vos paramètres. Assurez-vous que c'est bien ce que vous souhaitez avant de confirmer.
