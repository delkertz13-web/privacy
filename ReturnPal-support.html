Questions Fréquemment Posées

Comment ajouter un achat dans ReturnPal ?
Appuyez sur le bouton "+" dans l'onglet Accueil, sélectionnez votre revendeur dans la bibliothèque préremplie (ou ajoutez-le manuellement), entrez l'article, la date d'achat et la durée de retour. ReturnPal calculera automatiquement la date limite de retour.
Comment fonctionnent les notifications ?
Vous recevrez des rappels intelligents à 7 jours, 3 jours et 1 jour avant l'expiration de votre fenêtre de retour. Chaque notification inclut un lien direct vers l'app du revendeur pour un retour facile.
Quelle est la différence entre la version gratuite et Pro ?
La version gratuite permet de suivre jusqu'à 10 achats actifs simultanés avec une bibliothèque de 50+ revendeurs. La version Pro (1,99 € achat unique) offre un suivi illimité, la bibliothèque complète de revendeurs et un historique archivé.
Mes données sont-elles sécurisées ?
Oui, toutes vos données sont stockées localement sur votre appareil. ReturnPal ne collecte jamais vos informations d'achat ou de paiement. Votre vie privée est notre priorité.
Puis-je restaurer mes achats sur un nouvel appareil ?
Utilisez la fonction "Restaurer les achats" pour synchroniser vos données via iCloud. Si vous n'utilisez pas iCloud, vous pouvez exporter vos données pour les importer ultérieurement.

Restaurer les Achats

Si vous avez précédemment acheté ReturnPal Pro ou si vous avez changé d'appareil, utilisez cette option pour restaurer votre accès à la version Premium.

Vous serez redirigé vers l'authentification App Store pour vérifier votre achat précédent.

Supprimer les Données

Vous pouvez supprimer toutes vos données ReturnPal à tout moment. Cette action est irréversible et supprimera tous vos achats enregistrés et vos paramètres.

⚠️ Attention : La suppression de vos données ne peut pas être annulée. Assurez-vous que vous n'avez plus besoin de vos retours suivis avant de continuer.
