❓ Questions Fréquemment Posées

Comment configurer la durée de ma session de travail ?
Ouvrez l'app et accédez à Paramètres (roue dentée). Vous pouvez choisir entre 3 profils prédéfinis : 25 minutes (Pomodoro), 50 minutes (session standard) ou 90 minutes (session longue). Vous pouvez également créer un profil personnalisé en saisissant le nombre de minutes souhaité. Votre choix sera automatiquement appliqué au prochain timer lancé.
Puis-je ignorer une pause posturale ?
Les pauses sont recommandées mais pas obligatoires. Quand votre session arrive à terme, PostureBreak affiche une notification avec l'exercice du jour. Vous pouvez ignorer cette pause, mais votre compliance hebdomadaire sera affectée. Pour ignorer sans notification, vous pouvez aussi utiliser la fonction "Ignorer cette pause" dans l'app pour continuer à travailler.
Comment fonctionne le mode Focus avec détection idle ?
Si vous activez "Mode Focus" dans les paramètres, PostureBreak utilise les capteurs de mouvement de votre iPhone pour détecter si vous êtes inactif sur votre Mac. Si l'app détecte que vous n'avez pas bougé pendant 2 minutes, elle génère une notification plus intense pour vous rappeler de faire une pause. Cette fonction nécessite l'accès aux données de mouvement et peut être désactivée à tout moment.
Qu'est-ce qui est inclus dans la version gratuite ?
La version gratuite inclut le timer de session, 3 exercices d'étirement illustrés (cou, épaules, dos), un profil utilisateur et les notifications de base. L'achat unique à 1,99 $ déverrouille la bibliothèque complète de 12 exercices, les statistiques avancées de compliance, les streaks de bonnes journées, et un widget iOS pour votre écran d'accueil.
Pourquoi mon historique de compliance a disparu après une réinstallation ?
Les données de compliance sont stockées localement sur votre appareil. Si vous réinstallez l'app, les données historiques ne peuvent pas être récupérées automatiquement. Cependant, si vous utilisez iCloud Sync (activable dans les paramètres), vos données seront synchronisées et restaurées après la réinstallation. Nous recommandons d'activer iCloud Sync pour protéger votre historique.

Si vous avez déjà acheté l'accès complet sur cet appareil ou un autre, vous pouvez restaurer votre achat sans frais supplémentaires.

Si vous souhaitez supprimer complètement vos données de PostureBreak, y compris votre historique de compliance, vos paramètres et vos préférences, vous pouvez utiliser l'option ci-dessous.

⚠️ Cette action est irréversible. Tous vos données, y compris votre historique de pauses et votre compliance tracking, seront supprimées de manière permanente. Veuillez confirmer avant de procéder.

Si vous utilisez iCloud Sync, vous aurez la possibilité de désactiver la synchronisation ou de supprimer également les données du cloud. Cette action ne supprimera pas votre achat et vous pourrez réactiver votre profil avec les mêmes fonctionnalités.
