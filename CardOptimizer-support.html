Questions Fréquemment Posées

Comment ajouter mes cartes de crédit dans CardOptimizer?

CardOptimizer fonctionne hors ligne et ne stocke jamais vos données bancaires. Pour ajouter vos cartes:

  1. Ouvrez l'onglet "Mes Cartes"
  2. Appuyez sur le bouton "+"
  3. Sélectionnez votre type de carte (Visa, Mastercard, etc.)
  4. Les taux de cashback sont automatiquement remplis pour les 50+ cartes pré-chargées
  5. Vos données restent uniquement sur votre téléphone
Comment utiliser le Scanner d'Achat pour optimiser mes récompenses?

Le Scanner d'Achat vous aide à choisir la meilleure carte instantanément:

  1. Ouvrez l'onglet "Scanner"
  2. Sélectionnez la catégorie d'achat (alimentation, carburant, voyage, etc.)
  3. Entrez le montant de votre achat
  4. L'app affiche quelle carte vous donne le meilleur cashback
  5. Vous voyez aussi le gain estimé en points
Les données de mes cartes sont-elles sécurisées?

Oui, vos données sont complètement sécurisées. CardOptimizer:

  • Fonctionne entièrement hors ligne
  • Ne se connecte jamais à Internet pour vos données personnelles
  • Ne stocke vos informations que localement sur votre téléphone
  • Ne demande jamais vos numéros de carte complets
  • N'envoie aucune donnée à des serveurs externes
Que signifie le Dashboard et comment l'interpréter?

Le Dashboard affiche vos statistiques de récompenses:

  • Récompenses Gagnées: Points et cashback que vous avez reçus
  • Récompenses Perdues: Points que vous auriez pu gagner avec la meilleure carte
  • Gain Potentiel: Argent que vous pourriez économiser par mois
  • Ces statistiques vous aident à voir l'impact réel d'optimiser vos cartes
Comment puis-je exporter ou sauvegarder mes données de cartes?

Pour exporter vos données et les sauvegarder de manière sécurisée:

  1. Allez dans l'onglet "Paramètres"
  2. Appuyez sur "Exporter les Données"
  3. Vos données de cartes seront envoyées à votre email sous forme sécurisée
  4. Vous pourrez les importer dans une nouvelle installation de l'app
  5. Cette fonctionnalité est disponible dans la version Premium

Restaurer Votre Achat

Si vous avez acheté la version Premium de CardOptimizer sur un autre appareil ou après une réinstallation, vous pouvez restaurer votre achat.

Comment restaurer votre achat:

  1. Ouvrez CardOptimizer
  2. Allez dans l'onglet "Paramètres"
  3. Appuyez sur le bouton "Restaurer les Achats"
  4. Identifiez-vous avec votre compte Apple ID
  5. L'app vérifiera vos achats précédents
  6. Vous recevrez un message de confirmation
✓ Achat restauré avec succès! La version Premium est maintenant activée.

Supprimer Vos Données

CardOptimizer respecte votre vie privée. Vous pouvez à tout moment supprimer toutes vos données personnelles stockées sur votre appareil.

Que sera supprimé:

  • Toutes les cartes de crédit que vous avez ajoutées
  • Tous les historiques d'achats et transactions
  • Vos paramètres personnalisés
  • Vos statistiques de récompenses

Que ne sera PAS supprimé:

  • Votre achat Premium (stocké dans les serveurs Apple)
  • Les données de votre compte Apple ID
⚠️ Attention: Cette action est irréversible. Assurez-vous que vous avez sauvegardé vos données importantes avant de continuer.

Comment supprimer vos données:

  1. Ouvrez CardOptimizer
  2. Allez dans l'onglet "Paramètres"
  3. Faites défiler vers le bas
  4. Appuyez sur "Supprimer Toutes les Données"
  5. Confirmez l'action (impossible à annuler)

Besoin de nous contacter avant de supprimer? Envoyez-nous un email si vous avez des questions.

✓ Toutes vos données ont été supprimées. CardOptimizer est maintenant réinitialisé.
